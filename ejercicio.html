Práctica con elementos de bloque y en línea

Elementos en bloque

Elementos que funcionan de forma independiente. Si no se indica lo contrario, generan un salto de línea.

address: aporta información de contacto para el elemento article más cercano o ancestro. También se puede aplicar al elemento body (a todo el documento).
article: representa un contenido independiente o reutilizable, puede ser un artículo, una entrada de un blog, un comentario de un usuario, o cualquier elemento independiente del contenido.
aside: representa una seccion que está indirectamente relacionada con el contenido principal, se usa principalmente para los contenidos de las barras laterales, publicidad, información de perfil, enlaces relacionados, etc.

audio: se utiliza para insertar un audio mediante un archivo o un enlace.

blockquote: crea citas en bloque, marca las citas a otros autores o documentos.

canvas: se utiliza para dibujar gráficos a través de secuencias de comandos.

.
div: sirve para crear secciones o agrupar contenidos.
dl: representa una lista de términos dt y de descripciones con elementos dl.
dt: muestra un término agrupado en una lista dl y con una definición dd.
dd: aporta detalles o una definición sobre un elemento dt en una lista de descripciones dl.
fieldset: permite organizar en grupos los campos de un formulario.
footer: representa un pie de página.
form: representa una sección con controles interactivos que permiten enviar información a un servidor.

h1-h6: representa 6 niveles de cabecera o títulos.

header: representa una sección de cabecera que puede contener elementos de navegación, logotipos, busquedas, etc.
hgroup: representa el encabezado de una sección agrupando cabeceras h1-h6.

hr/: sólo se usa una etiqueta y representa un cambio de tema entre párrafos, muestra una línea divisoria:

    nav: crea un menú de navegación, dentro de él se añade una lista ul.

    noscript: aporta contenidos alternativos al alemento script, en caso de no que la aplicación no soporte el script, carga el contenido almacenado en noscript.

      ol: crea una lista de elementos li ordenada.
    output: representa el resultado de un cálculo

    p: elemento de párrafo. ...

    section: representa una sección genérica de un documento.
    table: representa datos en dos o más dimensiones. Usa elementos tr para almacenar una fila, th para los nombres de cabecera, td para los diferentes campos de una fila. Se pueden agrupar columnas con el elemento colgroup.
    tfoot: representa un pie de tabla.
      ul: crea una lista de elementos li no ordenada.

    video: se utiliza para insertar un video mediante un archivo o un enlace.

    Elementos en línea

    Son elementos que se pueden utilizar dentro de un texto, para dotar de significado o formato a una parte de este o si se le indica, como un bloque. Si no se le indica, no genera un salto de línea.

    a: crea un enlace otra páginna, a un elemento de la misma, a una direccion de correo, teléfono o cualquier URL.
    abbr: representa la abreviación de un término.
    acronym: representa un acróniomo.
    b: muestra el texto en negrita (bold en inglés.
    bdo: permite modificar el orden de lectura lectura.
    big: muestra el texto con un tamaño de fuente mayor.
    br: genera un salto de linea en el texto, sólo tiene una etiqueta.
    muestra un botón con el que el usuario puede interactuar.
    cite: marca una referencia a una fuente, o el autor de un texto citado.
    code: marca el código de un programa.
    dfn: sirve para marcar el término que se quiere definir.
    em: sirve para marcar con énfasis las partes importantes de un texto.
    i: muestra el texto en cursiva (itálica).
    img: representa una imagen en el documento.
    input: se usa para crear controles interactivos que reciben datos del usuario. Su comportamiento depende del atributo type (button, checkbox, color, date, datetime, dateime-local, email, file, hidden, image, month, number, password, radio, range, reset, search, submit, tel, text, time, url, week)
    kdb: marca el texto que tiene que introducir el usuario o la acción a realizar.
    representa una etiqueta para otro elemento, se puede asociar con el atributo for o situando el elemento dentro de elemento label.
    map: asociado a una imagen, sirve para convertir ciertas areas de dicha imagen en enlaces
    objetct: define un objeto insertado en la página, puede cualquier elemento mutimedia(audio, video, Java applets, ActiveX, pdf, Flash u otra web). Permite pasar parámetros a plugings embebidos dentro del elemento. q: crea citas en linea, marca citas a otros autores o documentos.
    samp: marca la salida o resultado de un programa.
    script: inserta o hace referencia a un script ejecutable.
    select: muestra un elemento desplegable en el que se pueden elegir varias opciones:
    small: muestra el texto con un tamaño de fuente menor.
    span: es un contenedor en linea, sirve para agrupar elementos en linea o aplicar estilo al texto.
    strong: sirve para marcar énfasis a partes importantes del texto.
    sub: sirve para mostrar subíndices.
    sup: sirve para mostrar superíndices.
    control para editar texto plano con varias líneas.
    time: sirve para representar una hora o fecha. 20:30
    tt: muestra el texto con una fuente de ancho fijo
    var: marca variables de programas.
